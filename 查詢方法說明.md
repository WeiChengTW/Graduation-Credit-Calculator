# 長庚大學課程學分查詢方法

## 目標
從長庚 MOOCS 取得學生修課清單，再到課程查詢系統抓取每門課的詳細資料（學分、授課教師等）。

---

## Step 1：從 MOOCS 取得修課清單

**網站：** https://moocs.cgu.edu.tw/learn/index.php

登入後，課程列表藏在 frameset 裡。直接瀏覽此網址可取得課程列表：

```
https://moocs.cgu.edu.tw/learn/mycourse/index.php
```

每頁 10 筆，分頁用 form POST 控制：

```javascript
// 跳到第 N 頁
const form = document.querySelector('form#actFm');
form.page.value = 'N';
form.submit();
```

取得每頁課程名稱（含開課序號）：

```javascript
Array.from(document.querySelectorAll('table tr td:first-child a'))
  .map(a => a.textContent.trim())
```

課程名稱格式範例：`114-2-人工智慧-A2342`
- `114` = 學年
- `2` = 學期
- `人工智慧` = 課程名稱
- `A2342` = 開課序號（下一步查詢用）

---

## Step 2：從課程查詢系統取得詳細資料

**API 端點：**

```
GET https://catalog.cgu.edu.tw/IsService/api/Course/GetCourseSections
```

**參數：**

| 參數 | 說明 | 範例 |
|------|------|------|
| termid | 學期代碼（見下表） | 70 |
| sectionid | 開課序號 | A2475 |
| 其餘參數 | 留空即可 | |

**學期代碼對照表：**

| 學年學期 | termid |
|----------|--------|
| 114 / 2  | 70 |
| 114 / 1  | 69 |
| 113 / 2  | 67 |
| 113 / 1  | 66 |
| 112 / 2  | 64 |
| 112 / 1  | 63 |
| 112 / 3（暑）| 65 |
| 113 / 3（暑）| 68 |
| 114 / 3（暑）| 71 |

**完整 curl 範例：**

```bash
curl "https://catalog.cgu.edu.tw/IsService/api/Course/GetCourseSections?termid=70&departmentid=&call_id=&keyward=&sectionid=A2475&teaName=&cName=&year=&fieldid=&week=&stime=&etime="
```

**回傳 JSON 重要欄位：**

| 欄位 | 說明 |
|------|------|
| ACADMICYEAR | 學年 |
| ACADMICTERM | 學期 |
| CALL_ID | 科目代號 |
| SECTIONID | 開課序號 |
| DEPARTMENTNAME_C | 開課單位 |
| YEAR | 年級 |
| CCOURSENAME | 課程名稱（中文） |
| NAME | 授課教師 |
| CREDITS | 學分數 |
| C_PTIME | 上課時間（中文格式，例：一 18:10-20:00） |
| CLASSIFICATIONCATNAME | 課程類別（必修/選修等） |

## Step 3：Python 批次查詢範例

```python
import urllib.request
import json
import csv

API = "https://catalog.cgu.edu.tw/IsService/api/Course/GetCourseSections"

# 格式：(termid, 開課序號)
courses = [
    (70, "A2475"),
    (70, "A2342"),
    # ... 其餘課程
]

rows = []
total_credits = 0

for termid, sectionid in courses:
    url = f"{API}?termid={termid}&departmentid=&call_id=&keyward=&sectionid={sectionid}&teaName=&cName=&year=&fieldid=&week=&stime=&etime="
    with urllib.request.urlopen(url) as r:
        data = json.loads(r.read())[0]

    credits = data.get('CREDITS', 0)
    total_credits += credits

    rows.append([
        f"{data['ACADMICYEAR']}-{data['ACADMICTERM']}",
        data.get('CALL_ID', ''),
        sectionid,
        data.get('DEPARTMENTNAME_C', ''),
        data.get('YEAR', ''),
        data.get('CCOURSENAME', ''),
        data.get('NAME', ''),
        credits,
        data.get('C_PTIME', ''),
        data.get('CLASSIFICATIONCATNAME', ''),
    ])

# 輸出 CSV
with open('courses_detail.csv', 'w', newline='', encoding='utf-8-sig') as f:
    writer = csv.writer(f)
    writer.writerow(['學年學期','科目代號','開課序號','開課單位','年級','課程名稱','授課教師','學分','上課時間','課程類別'])
    writer.writerows(rows)

print(f"共 {len(rows)} 門課，總學分：{total_credits}")
```

---

## 注意事項

1. **MOOCS 登入**：需要學號與密碼，登入後 session 保持在瀏覽器中。
2. **catalog.cgu.edu.tw 不需登入**：API 為公開查詢，直接呼叫即可。
3. **體育課學分為 0**：`CREDITS` 欄位會回傳 0.0，需注意是否要另計。
4. **開課序號格式差異**：114 學年第 2 學期用英數混合（如 A2475），其餘學期用純數字（如 71798）。
